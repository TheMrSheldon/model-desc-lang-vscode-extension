# VSCode Model Description Language Extension

## Installation
### From Nightly
Download [the latest nightly build](https://github.com/TheMrSheldon/model-desc-lang-vscode-extension/releases/download/nightly/mdl-extension.vsix) and install it through `code install <path-to-vsix>`. For example, on Linux you may use
```bash
wget https://github.com/TheMrSheldon/model-desc-lang-vscode-extension/releases/download/nightly/mdl-extension.vsix
code install mdl-extension.vsix
rm mdl-extension.vsix
```
